# docs/decisions · 决策记录

> 记录关键产品/技术决策（Architecture Decision Records, ADR）。

---

## 用途

当做出影响架构、交互、数据模型或第三方选型的关键决策时，在此归档简要记录，便于新成员快速理解「为什么这样设计」。

## 记录格式

每个决策建议包含：

- **决策名称**
- **背景**
- **考虑过的选项**
- **最终决策**
- **理由**
- **影响范围**
- **相关文档链接**
