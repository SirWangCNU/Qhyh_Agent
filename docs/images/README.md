# docs/images · 图片资料目录

> 存放截图、界面参考图、流程图、架构图等产品相关资料。

---

## 目录用途

| 子目录 | 用途 |
|--------|------|
| `screenshots/` | 前端页面截图、功能演示图 |
| `wireframes/` | 线框图、原型图 |
| `architecture/` | 系统架构图、数据流图 |
| `references/` | 竞品参考图、设计参考图 |
| `prompts/` | 与提示词工程相关的示意图片 |

---

## 图片命名规范

建议采用以下格式：

```
{模块}-{描述}-{日期}.{扩展名}
```

例如：

- `workshop-step-list-20250629.png`
- `architecture-pipeline-20250629.png`
- `homepage-sidebar-20250629.png`
